========================================
ملخص سريع عن لغة HTML
========================================

ما هي HTML؟
------------
HTML اختصار لـ HyperText Markup Language، وهي لغة الترميز الأساسية لبناء صفحات الويب.
هي مش لغة برمجة بالمعنى التقليدي (زي بايثون أو جافا)، لكنها لغة "توصيف" بتحدد شكل ومحتوى الصفحة اللي بتظهر في المتصفح.

فايدة HTML
------------
- هي الهيكل العظمي لأي صفحة ويب (بتحدد العناوين، الفقرات، الصور، الروابط، الجداول... إلخ).
- بتشتغل مع كل المتصفحات وعلى أي جهاز.
- سهلة التعلم وأساس لازم لأي حد عايز يدخل مجال الويب.
- بتتكامل مع CSS (للتنسيق والشكل) و JavaScript (للتفاعل والحركة) عشان تطلع صفحة ويب كاملة.
- منظمة من جهة عالمية (W3C) فبتضمن توافق ومعايير موحدة.

أهم حاجات في اللغة
--------------------
1. العناصر (Elements): بتتكون من Tag فاتح و Tag قافل، زي: <p>نص</p>
2. الخصائص (Attributes): بتضيف معلومات إضافية للعنصر، زي: <img src="pic.jpg" alt="صورة">
3. هيكل الصفحة الأساسي:
   <!DOCTYPE html>
   <html>
     <head> ... </head>
     <body> ... </body>
   </html>
4. الترتيب الهرمي (Nesting): العناصر ممكن تتحط جوه بعض بشكل منظم.
5. الدلالية (Semantic HTML): استخدام Tags بمعنى واضح (زي <header>, <footer>) بيحسن الـ SEO ويسهل الوصول (Accessibility).

أشهر الـ Tags استخداماً
--------------------------
<html>          - العنصر الرئيسي اللي بيحتوي الصفحة كلها
<head>          - بيحتوي معلومات عن الصفحة (العنوان، الروابط، الميتاداتا)
<title>         - عنوان الصفحة اللي بيظهر في التاب
<body>          - بيحتوي كل المحتوى المرئي في الصفحة
<h1> إلى <h6>   - العناوين بترتيب الأهمية
<p>             - فقرة نصية
<a>             - رابط (Hyperlink)
<img>           - صورة
<div>           - حاوية عامة لتقسيم المحتوى
<span>          - حاوية صغيرة داخل السطر لتنسيق جزء من النص
<ul> / <ol>     - قائمة غير مرقمة / مرقمة
<li>            - عنصر داخل القائمة
<table>         - جدول
<tr> / <td>     - صف / خلية في الجدول
<form>          - نموذج لإدخال البيانات
<input>         - حقل إدخال بيانات
<button>        - زرار
<header>        - رأس الصفحة أو القسم
<footer>        - تذييل الصفحة أو القسم
<nav>           - قائمة التنقل
<section>       - قسم في الصفحة
<article>       - محتوى مستقل زي مقال أو بوست
