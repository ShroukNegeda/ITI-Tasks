About

I'm a Cairo-based developer who spends most days somewhere between at the keyboard and a code editor. I studied visual communication before teaching myself to build the interfaces that make it all work.

My favourite projects start messy — sticky notes, half-finished wireframes, a dozen font pairings that all get thrown away — and end with something that feels obvious in hindsight. I care about legible type, generous spacing, and interfaces that don't make people think twice.

Outside of client work, I run a small newsletter about Egyptian typography, collect old travel posters, and I'm slowly redesigning my own portfolio for what feels like the tenth time this year.

I like calm colour palettes, honest copywriting, and CSS that does exactly what it says on the tin — nothing sneaky, nothing bloated.
